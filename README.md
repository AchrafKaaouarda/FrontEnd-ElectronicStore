# Electronics Store

This project is a simple electronics store application built with React. It showcases a variety of electronic products and allows users to add items to their cart.

## Features

- Display a list of electronic products with details such as title, description, price, and image.
- Add products to the shopping cart.
- View items in the cart and proceed to checkout.
- Responsive design for a better user experience.

## Technologies Used

- React
- CSS
- JavaScript

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/AchrafKaaouarda/FrontEnd-ElectronicStore
   ```

2. Navigate to the project directory:
   ```
   cd electronic-store
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.
